Notes on component definition for ESP-IDF stable (v3.3.x) and ESP-IDF v4.x

Minimal CMakeLists.txt file:

ESP-IDF stable (v3.3.x)

set(COMPONENT_SRCS "foo.c" "bar.c") set(COMPONENT_ADD_INCLUDEDIRS ".") register_component()

ESP-IDF v4.x

idf_component_register(SRCS "foo.c" "bar.c" INCLUDE_DIRS ".")

If the component needs other components only for compiling it add them as private required

ESP-IDF stable (v3.3.x)
set(COMPONENT_SRCS "foo.c" "bar.c") set(COMPONENT_ADD_INCLUDEDIRS ".") set(COMPONENT_PRIV_REQUIRES componentA componentB) register_component()
ESP-IDF v4.x
idf_component_register(SRCS "foo.c" "bar.c" INCLUDE_DIRS "." PRIV_REQUIRES componentA componentB)

else add them as reqired and will be available for the entire project
ESP-IDF stable (v3.3.x)
set(COMPONENT_SRCS "foo.c" "bar.c") set(COMPONENT_ADD_INCLUDEDIRS ".") set(COMPONENT_PRIV_REQUIRES componentA componentB) set(COMPONENT_REQUIRES componentC componentD) register_component()
ESP-IDF v4.x
idf_component_register(SRCS "foo.c" "bar.c" INCLUDE_DIRS "." PRIV_REQUIRES componentA componentB REQUIRES componentC componentD)
